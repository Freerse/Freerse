## Dependencies

- Xcode 15.x
- iOS SDK 16.x

## Setup dev environment

1. Clone this repository
2. Set up flutter devlopment 
3. Enter API keys in Config.xcconfig. 
4. Open Freerse.xcodeproj in Xcode
5. Wait for Package Dependencies to finish downloading
6. Run
